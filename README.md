              ----- 📊 Superstore Sales Analysis using SQL  ------

## Project Description

This project performs sales and profit analysis on a Superstore dataset using SQL.
The objective is to extract meaningful business insights such as top-performing products, regional performance, customer behavior, and 
category-wise profitability.

This project demonstrates strong SQL skills including joins, aggregations, subqueries, and window functions.


---  Objectives---
1. Analyze sales performance
2. Identify top selling products
3. Find most profitable categories
4. Analyze region-wise performance
5. Identify top customers
6. Detect loss-making orders
7. Monthly sales trend analysis

--- Tables Used---
The project is built using the following tables:
* Customers
* Orders
* Order_Details
* Products
* Location

## SQL Analysis Performed
--- Basic Queries----
* Filter customers by segment
* Find orders after specific date
* Products with discount
* Orders by ship mode

--- Joins

* Customer name with orders
* Product name with sales
* Orders with region and city
* Customer orders with ship mode

---- Aggregations

* Total sales by category
* Profit by region
* Top 10 products by sales
* Category-wise profit

---- Advanced SQL

* Customers who never ordered
* Orders with negative profit
* Running total of sales
* Monthly sales trend
* Second highest sales product
* Top customer in each region
* Highest profit category
